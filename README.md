# Club

Club is a web-app for managing guests at a club.

## Installation

* `git clone <https://github.com/klinkow/Club>` this repository
* `cd Club`
* `npm install`
* `bower install`

## Running / Development

* `ng build`
* `ng serve`
* Visit your app at [http://localhost:4200](http://localhost:4200).
